# dbt_github_actions_demo

## Prerequisites

* Must create the following Git secrets:

* `SNOWFLAKE_ACCOUNT`
* `SNOWFLAKE_USER`
* `SNOWFLAKE_PASS`
* `SNOWFLAKE_ROLE`
